Classic Game

Play the Original

Step back to 1983! A two-player challenge where the goal is simple: survive longer than your opponent.

Board starts yellow with a black border. Navigate into empty spaces and avoid the border, the other player, and yourself (see Practice Mode below).

Technical Details and History

This original DOM-based version uses standard HTML elements and the long-deprecated document.write. Written in JavaScript around 1998, it was inspired by my first 1983 version on a PDP-11, written in Coral-66 and run from a VT100 terminal.

The name "Domino" comes from the terminal screen looking like domino tiles being laid.

Keeping this original for simplicity of code readability for newcomers to programming. See plans for new 2025 version below
