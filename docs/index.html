Welcome to Domino 2025

The original game

Play the original game

This is the classic version of the game from 1983, designed for 2 players. The object of the game is to keep going longer than your opponent.

The board starts yellow with a black border. Each player must steer themselves into blank (yellow) space and avoid hitting the border, the other player, or themselves.

Player 1 Q W
Player 2 O P


Technical details and history

This classic DOM-based version uses standard HTML elements and document.write. It was written in JavaScript around 1998, based on my very first version from 1983 on a PDP-11 using ANSI escape sequences.

The name "Domino" was inspired by the character-based terminal screen that resembled domino tiles being laid.

I am keeping the original for simplicity and plan to use it as a base to allow more players and transition to HTML Canvas, eventually removing document.write.
