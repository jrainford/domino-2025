The original game

This is the classic, but outdated, DOM-based version built years ago. It uses standard HTML elements (TABLE, TR, TD) rather than canvas. It also relies on document.write, which is long deprecated. It was written in JavaScript around 1998. Going back even further, I created the very first version of "domino" in 1983 on a PDP-11 in Coral-66 running on a VT100 terminal using ANSI escape sequences.
