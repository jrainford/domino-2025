The original game

The original game

This is the classic version of the game from 1983.

The game is for 2 players. The object of the game is to keep going longer than your opponent.

The board starts out yellow with a black border. Each player must steer themselves into blank, ie yellow space. They must not hit the border, the other player or themselves.

Player controls:
Player Turn Left Turn Right
Player 1 Q W
Player 2 O P



Technical details and history

This is the classic, but outdated, DOM-based version built years ago. It uses standard HTML elements (TABLE, TR, TD) rather than canvas. It also relies on document.write, which is long deprecated. It was written in JavaScript around 1998. Going back even further, I created the very first version of "domino" in 1983 on a PDP-11 in Coral-66 running on a VT100 terminal using ANSI escape sequences. This 1998 Javascript version plays the same as on the 1983 VT100 version.

The name "Domino" came to me back in 1998 as the character-based nature of the terminal screen made it look like domino tiles being laid.

I am keeping the original as it's uncomplicated by objects, jquery, events etc, so easy to explain to someone who's never seen programming of any kind. I plan to use the original as a base to include perhaps simple player objects in an array to allow multi-player (rather than just 2) and to use the HTML Canvas. I plan to ditch use of document.write
